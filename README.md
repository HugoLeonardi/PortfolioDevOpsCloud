### Infrastructure 3-Tiers Nextcloud sur AWS
Déploiement automatisé de Nextcloud sur AWS avec Terraform.
Architecture 3-tiers : EC2 en subnet public, RDS MySQL en subnets privés, stockage S3 découplé. Sécurité intégrée dès la conception, déploiement en une commande.

**Stack :** Terraform · AWS EC2 · RDS MySQL · S3 · VPC · IAM
